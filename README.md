# Lucas Nascimento · Parrillero

A showcase website for Lucas Nascimento, a parrillero (barbecue chef) based in Cascavel, Brazil.

**Built with:** HTML · CSS · JavaScript

## About

A single-page showcase presenting the chef's work, style and contact, built with plain HTML, CSS and JavaScript for fast loading and simple hosting.

## Running locally

Open `index.html` in your browser, or serve the folder with any static server.
